REST API - клиент отправляет запрос на сервер в виде данных. Сервер использует этот клинтский ввод для запуска внутренних функций и возвращает выходные данные обратно клиенту. Определяет набор функций таких как GET, PUT, DELETE и тд., которые клиенты могут использовать для доступа к данным сервера. Клиент и сервер обмениваются данными по протоколу HTTP
